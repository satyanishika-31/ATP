# Week 1 JavaScript Practice Scripts

This folder contains small JavaScript programs that demonstrate basic logic using numbers, arrays, and object manipulation. Each file focuses on one simple task.

## File List

### 1. `Biggest_of_2Numbers.js`
Finds the bigger value between two numbers using an `if/else` condition.

### 2. `Biggest_of_3Numbers.js`
Finds the biggest value among three numbers using `if`, `else if`, and `else`.

### 3. `FindingBigAmong_3numbers(arr).js`
Defines a function that takes an array and returns the largest number in it.

### 4. `FIndingSmallAmongArr.js`
Finds the smallest value in an array of marks.

### 5. `OperationsWithEmpDetails.js`
Performs array operations on employee data:
- Inserts a new employee at the 2nd position
- Removes the employee named `Kiran`
- Changes the last mark of `Sneha` from `95` to `75`

### 6. `serarchingElementInArr.js`
Searches for an element inside an array and returns its index. If the value is not present, it returns `not found`.

### 7. `sumOfAnArr.js`
Calculates the sum of all numbers in an array.

## What This Folder Demonstrates

- Basic `if/else` logic
- Looping through arrays
- Function creation and return values
- Simple search and sum operations
- Working with objects and nested arrays
