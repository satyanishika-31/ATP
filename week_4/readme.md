# Week 4 HTML/CSS Projects

This folder contains a set of small practice projects built with HTML and CSS. Each folder is mostly independent and can be opened directly in a browser by starting from its main HTML file.

## Project List

### 1. `basic_web_Page`
Main files:
- `bang.html`
- `cha.html`
- `city.html`
- `hyb.html`
- `style.css`
- `city.css`

Description:
- A city showcase project focused on Indian cities.
- `city.html` works like a landing page and links to the city-specific pages.
- `hyb.html` describes Hyderabad.
- `cha.html` describes Chennai.
- `bang.html` describes Bangalore.
- The pages use card-style layouts, images, and shared styling.

### 2. `BLACK GOOSE BISTOR`
Main files:
- `ass_4.html`
- `style.css`

Description:
- A simple restaurant-style page inspired by the Black Goose Bistro layout.
- It uses a centered content card, image header, headings, divider line, and short text sections.

### 3. `full_web_page`
Main files:
- `index.html`
- `style.css`

Description:
- A blog or portfolio-style web page called QuickBlog.
- It includes a navigation bar, article cards, buttons, and a footer section.
- The CSS includes responsive behavior for smaller screens.

### 4. `html`
Main file:
- `ass_1.html`

Description:
- A beginner HTML practice page.
- It includes a heading, paragraphs, and an unordered list.

### 5. `HTML_CARD`
Main files:
- `card.html`
- `style.css`

Description:
- A profile card page with decorative shapes and a centered card design.
- The content introduces a Copilot-style profile card with a button.

### 6. `html_web_bage`
Main files:
- `ass_3.html`
- `style1.css`

Description:
- A simple profile-style page for Tom the cat.
- It includes a heading, image, introduction, profile list, links, and a short paragraph.
- The CSS gives the page a light blue background and floats the image to the right.

## How To Use

1. Open the folder of the page you want to view.
2. Start with the main HTML file listed above.
3. Open the HTML file in a browser or use Live Server in VS Code.
4. The linked CSS file will load automatically if it is in the same folder.

## Notes

- Some folder and file names contain spelling mistakes from the original practice work, but the files still function normally.
- Several pages use external image links and base64-encoded images, so internet access may be needed for the remote images to load.
- The pages are practice exercises, so the content is intentionally simple and mostly static.
