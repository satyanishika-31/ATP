# JavaScript Practice Projects — Week 2

This repository is a collection of small JavaScript exercises and mini-applications created for Week 2.

**Overview**
- **Purpose:** Practice common JavaScript patterns (arrays, objects, timers, simple apps).
- **Languages:** JavaScript (Node.js-compatible files)

**Project Structure**

- Root utilities and exercises:
  - BankTransactionAnalyzer.js — analyze transactions and balances
  - CopyExtendArray.js — array copying/extension examples
  - DailyTemperatureAnalyzer.js — temperature stats
  - EmployeePayrollProcessor.js — payroll calculation examples
  - MovieStreamingPlatform.js — basic streaming platform logic
  - Online CourseNameProcessor.js — course name processing utilities
  - recordingTemperature.js — temperature recording examples
  - restParameter.js — examples using rest parameters
  - setInterval.js — demo using `setInterval`
  - setTimeout.js — demo using `setTimeout`
  - ShallowDeepCopy.js — shallow vs deep copy examples
  - shoppingcartsummary.js — shopping cart summary helper
  - StudentMarksList.js — student marks array handling
  - StudentPerformanceDashboard.js — student performance summary

- ProblemStatment/
  - LibaryManagementSystem.js — small library management example
  - EcommerceShopping/
    - app.js — simple ecommerce app entry
    - cart.js — cart logic
    - discount.js — discount rules
    - payment.js — payment flow stub
    - product.js — product model
  - TaskManagementSystem/
    - app.js — task management entry
    - task.js — task model
    - validator.js — validation helpers
