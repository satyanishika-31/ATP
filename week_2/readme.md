# Week 2 - JavaScript

This folder contains Week 2 JavaScript practice programs covering array methods, spread/rest operators, classes, timer functions, and simple module-based projects.

## Topics Covered

- Array methods: `filter()`, `map()`, `reduce()`, `find()`, `findIndex()`
- Spread operator and rest parameters
- Classes and objects
- Timer functions: `setTimeout()` and `setInterval()`
- ES module import/export
- Basic project structure

## Folders and Files

| Path | Description |
|---|---|
| `array_operations_simple/` | Simple array method assignments for temperatures, courses, and student marks. |
| `Array-operations-advanced/` | Advanced array object assignments for shopping cart, students, payroll, movies, and bank transactions. |
| `Shopping_cart/` | Modular shopping cart project with product, cart, discount, payment, and app files. |
| `Spread operator and Rest operator/` | Exercises using spread syntax, rest parameters, and object copying. |
| `task_managment/` | Simple task management project using modules and validators. |
| `Timer functions/` | Timer assignments for exam status and OTP countdown. |
| `class_demo.js` | Book class example for library book management. |
| `copy_extend_Array.js` | Array copying and extending using spread operator. |
| `employeeque.js.js` | Employee array operations like insert, remove, and update. |

## How to Run

Use Node.js to run any file:

```bash
node filename.js
```

For files inside folders, include the folder path:

```bash
node array_operations_simple/ass1dailyTemp.js
node Array-operations-advanced/ass5bank.js
node "Timer functions/timer_functionQ1.js"
```
